## インストール
以下のコマンドで必要なライブラリをインストールしてください。

```bash
pip install -U langchain
pip install hf_xet
pip install torch transformers accelerate langchain_community langchain_huggingface bitsandbytes
pip install pypdf
pip install sentence-transformers
```
